# NoirStock PWA v6.5.0

iPhone向け、オフライン前提の在庫・販売・ハンドメイド・出品先分析・申告補助PWAです。

## v6.5.0で追加
- 申告CSV一式の出力
- 申告入力補助表CSV
- 申告入力補助表HTML
- 売上明細CSV
- 販売手数料・送料CSV
- 経費明細CSV
- 在庫棚卸CSV
- 材料購入CSV
- 制作ロットCSV
- v6.4.4.1の分析タブ修正、v6.4.2のUIテーマ、v6.4.1.1のJSON Import修正は維持

## 出力場所
分析 → 申告モード → 申告CSV一式 / 申告入力補助表HTML
