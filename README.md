# NoirStock PWA v6.4.1

iPhone向け、オフライン前提の在庫・販売・ハンドメイド・確定申告補助PWAです。

## v6.4.1で追加
- 一般向けに見やすいウォーム系ダークUIトークン
- 出品先マスタ
- 出品履歴
- イベント管理
- 販売場所の保存
- 出品先別売上・利益の分析土台
- イベント別収支の分析土台
- 長期出品中の確認
- v6.4 JSON互換
